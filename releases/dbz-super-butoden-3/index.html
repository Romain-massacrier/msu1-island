Installation

Pour utiliser le pack MSU-1, appliquez le patch IPS sur la ROM propre indiquée ci-dessous, puis placez la ROM patchée dans le même dossier que le fichier .msu et les fichiers .pcm.

ROM attendue

Nom : Dragon Ball Z - Ultime Menace (France).sfc

Taille : 2 097 152 octets

Header : sans header

CRC32 : 593EEFE0

La ROM doit être propre, non modifiée et sans header.

  1. Téléchargez le pack MSU-1 depuis le dossier Google Drive.
  2. Appliquez le fichier IPS sur la ROM propre avec un outil comme Floating IPS ou Lunar IPS.
  3. Placez la ROM patchée dans le même dossier que le fichier .msu et les fichiers .pcm.
  4. Vérifiez que la ROM patchée, le fichier .msu et les fichiers .pcm utilisent le même nom de base.
  5. Lancez la ROM patchée avec un émulateur ou un flashcart compatible MSU-1.

Si les musiques MSU-1 ne se lancent pas, vérifiez d’abord le nom des fichiers. Le fichier .msu et les fichiers .pcm doivent porter le même nom de base que la ROM patchée.

Nom de base conseillé après patch : Dragon ball Z 3 msu

Exemple attendu :

  • Dragon ball Z 3 msu.sfc
  • Dragon ball Z 3 msu.msu
  • Dragon ball Z 3 msu-1.pcm
  • Dragon ball Z 3 msu-2.pcm
  • etc.
